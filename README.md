# Web-scraping-_1
This Python script uses Selenium WebDriver to scrape hospital information from JustDial's website, specifically targeting multispeciality hospitals in Nagpur. It automates a Chrome browser to navigate to the specified URL, waits for elements to load, and extracts key details (name, phone number, address, rating, and link) from each hospital listing. The scraped data is then saved to a CSV file named hospitals.csv with appropriate headers. The script handles potential errors gracefully by skipping problematic entries and prints the total number of hospitals scraped before closing the browser. Note that it requires ChromeDriver to be installed and available in the system's PATH for Selenium to function.
